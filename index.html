Mandelbrot Set Explorer
Explore the Mandelbrot set in real-time with dynamic zoom and palette animation. The explorer also visualises either the Julia set or an altered Mandelbrot set, where z0 is chosen based on your selected position.

Based on mandel.gart.nz with modifications.
